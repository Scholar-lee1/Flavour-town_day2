# Python Mini Projects Collection

## Overview

This repository contains a collection of simple Python programs demonstrating basic programming concepts and problem-solving skills.

Projects Included

- BMI Calculator ("bmi.py")
  Calculates Body Mass Index based on user input.

- Simple Calculator ("simple_calc.py")
  Performs basic arithmetic operations.

- Birthday Countdown ("bd_countdown.py")
  Calculates time remaining until a specific date.

- Dictionary Practice ("dict.py")
  Demonstrates usage of Python dictionaries.

- List Operations ("list.py")
  Demonstrates list manipulation in Python.

- Variables and Strings ("var_str.py")
  Covers basic variable and string operations.

---

## How to Run

1. Install Python (3.x)

2. Run any file using:
   python filename.py
   
   Example:
   python bmi.py

---

## Requirements

Most scripts use only built-in Python libraries.
If needed:
pip install -r requirements.txt
